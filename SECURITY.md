# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the BlueNexus OpenClaw plugin, please report it responsibly.

**Please do NOT open a public issue for security vulnerabilities.**

Instead, report security issues via:

- GitHub Security Advisories: https://github.com/bluenexus-ai/openclaw/security/advisories/new
- Email: security@bluenexus.ai

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

Only the latest published version receives security updates.
